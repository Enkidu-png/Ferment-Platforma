Add local type for admin product rows (alongside existing types):
```typescript
type AdminProductRow = {
  id: string;
  name: string;
  is_archived: boolean;
  created_at: string;
  tenant: { id: string; name: string; slug: string } | null;
};
```

Add to `productsRouter` (inside the `createTRPCRouter({...})` object, after existing procedures):

```typescript
adminGetProducts: adminProcedure
  .input(z.object({
    search: z.string().optional(),
  }))
  .query(async ({ input }) => {
    let query = supabaseAdmin
      .from("products")
      .select(
        "id, name, is_archived, created_at, " +
        "tenant:tenants!tenant_id(id, name, slug)"
      )
      .order("created_at", { ascending: false });

    if (input.search && input.search.trim() !== "") {
      query = query.ilike("name", `%${input.search.trim()}%`);
    }

    const { data, error } = await query;
    if (error) throw new Error(error.message);
    return (data ?? []) as unknown as AdminProductRow[];
  }),

adminArchiveProduct: adminProcedure
  .input(z.object({ productId: z.string() }))
  .mutation(async ({ input }) => {
    const { error } = await supabaseAdmin
      .from("products")
      .update({ is_archived: true })
      .eq("id", input.productId);
    if (error) throw new Error(error.message);
    return { success: true };
  }),

adminRestoreProduct: adminProcedure
  .input(z.object({ productId: z.string() }))
  .mutation(async ({ input }) => {
    const { error } = await supabaseAdmin
      .from("products")
      .update({ is_archived: false })
      .eq("id", input.productId);
    if (error) throw new Error(error.message);
    return { success: true };
  }),
```

Note: `adminGetProducts` uses `supabaseAdmin` — the existing `getMany` uses `ctx.supabase` (user-scoped + RLS). Admin must see ALL products across ALL tenants, including archived ones. The anon client filtered by RLS would only return the admin's own tenant products (or fail with 0 rows if admin has no tenant).

**Step A: ProductsView component**

Create `src/modules/admin/ui/views/products-view.tsx` — a `'use client'` component.

The view is a searchable table of all products with Archive/Restore buttons.

```typescript
'use client';
import { useState, useEffect } from "react";
import { trpc } from "@/trpc/client";
import { toast } from "sonner";
import {
  Table, TableBody, TableCell, TableHead, TableHeader, TableRow
} from "@/components/ui/table";
import { Button } from "@/components/ui/button";
import { Badge } from "@/components/ui/badge";
import { Input } from "@/components/ui/input";
```

State:
```typescript
const [search, setSearch] = useState("");
const [debouncedSearch, setDebouncedSearch] = useState("");

useEffect(() => {
  const timer = setTimeout(() => setDebouncedSearch(search), 300);
  return () => clearTimeout(timer);
}, [search]);
```

Data:
```typescript
const utils = trpc.useUtils();
const { data: products = [], isLoading } = trpc.products.adminGetProducts.useQuery(
  { search: debouncedSearch },
  { refetchOnWindowFocus: false }
);

const archiveMutation = trpc.products.adminArchiveProduct.useMutation({
  onSuccess: () => {
    utils.products.adminGetProducts.invalidate();
    toast.success("Product archived");
  },
  onError: (e) => toast.error(e.message),
});

const restoreMutation = trpc.products.adminRestoreProduct.useMutation({
  onSuccess: () => {
    utils.products.adminGetProducts.invalidate();
    toast.success("Product restored");
  },
  onError: (e) => toast.error(e.message),
});
```

Render:
```tsx
return (
  <div className="space-y-4">
    <Input
      placeholder="Search by product name..."
      value={search}
      onChange={(e) => setSearch(e.target.value)}
      className="max-w-sm"
    />
    {isLoading ? (
      <p className="text-muted-foreground">Loading products...</p>
    ) : products.length === 0 ? (
      <p className="text-muted-foreground">No products found.</p>
    ) : (
      <Table>
        <TableHeader>
          <TableRow>
            <TableHead>Product</TableHead>
            <TableHead>Merchant</TableHead>
            <TableHead>Status</TableHead>
            <TableHead>Created</TableHead>
            <TableHead className="text-right">Action</TableHead>
          </TableRow>
        </TableHeader>
        <TableBody>
          {products.map((product) => (
            <TableRow
              key={product.id}
              className={product.is_archived ? "opacity-50" : ""}
            >
              <TableCell className="font-medium">{product.name}</TableCell>
              <TableCell>{product.tenant?.name ?? "—"}</TableCell>
              <TableCell>
                <Badge variant={product.is_archived ? "secondary" : "default"}>
                  {product.is_archived ? "Archived" : "Active"}
                </Badge>
              </TableCell>
              <TableCell className="text-muted-foreground text-sm">
                {new Date(product.created_at).toLocaleDateString()}
              </TableCell>
              <TableCell className="text-right">
                {product.is_archived ? (
                  <Button
                    size="sm"
                    variant="outline"
                    onClick={() => restoreMutation.mutate({ productId: product.id })}
                    disabled={restoreMutation.isPending}
                  >
                    Restore
                  </Button>
                ) : (
                  <Button
                    size="sm"
                    variant="destructive"
                    onClick={() => archiveMutation.mutate({ productId: product.id })}
                    disabled={archiveMutation.isPending}
                  >
                    Archive
                  </Button>
                )}
              </TableCell>
            </TableRow>
          ))}
        </TableBody>
      </Table>
    )}
  </div>
);
```
